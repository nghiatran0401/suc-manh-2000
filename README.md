### How to run project locally

1. Client

- Go to `client`
- Install package dependency `npm install`
- Start frontend client `npm run client`

2. Server

- Go to `server`
- Install package dependency `npm install`
- Start backend server `npm run server`

3. Admin

- Go to `admin`
- Install package dependency `npm install`
- Start admin `npm run admin`
